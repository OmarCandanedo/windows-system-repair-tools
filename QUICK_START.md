## Windows Nuclear Repair v4.0 ULTIMATE

### ⚡ 30-Second Setup

1. **Download:** `WINDOWS_NUCLEAR_REPAIR_v4.0_ULTIMATE.bat`
2. **Right-click** → **Run as Administrator**
3. **Choose option** (usually **1** for complete repair)
4. **Wait** and restart PC when done
